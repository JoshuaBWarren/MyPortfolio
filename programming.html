Space Invaders

Project description

This game was part of a school program assignment. We had to create a Space Invaders type game in Racket with the following requirements:

  • Enemies that can shoot bullets down at the player
  • A player that can shoot bullets up at the enemies
  • Player has 3 lives and there is a life counter
  • Special enemy at the top that, if hit, grants one life
  • A way to score based on enemies hit
  • Game ends when all enemies are destroyed, if the player is destroyed, or if the enemies reach the bottom


    • The game has all the base requirements and was very fun to make!
